# sciadv

TUI cheatsheet for the Science Adventure series, written in rust

## Usage

```sh
sciadv -t chaos_head # more to be added soon
```

## Installation

### Build from source

```sh
git clone https://github.com/blootu/sciadv
cd sciadv
cargo build --release
```

### GitHub Releases

```
wget illfinishthislater 
```
